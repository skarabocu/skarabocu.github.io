My Projects

Multi-threaded Internet Search Engine
University of Michigan Capstone Course

Jan - Apr 2021

Ann Arbor, MI

Built an index (~200M websites and ~1 TB total size) including serialization/deserialization mechanisms for data compression with another teammate.
Implemented a constraint solver that will efficiently scan our entire index and find potential matching pages to user query.
Created a query compiler, which creates a logical relationship tree of terms in the user query that the constraint solver will use to scan web pages.
Designed a multi-threaded network communication channel between our ranker and tiny server to show the results to users.
Final search engine was distributed over 10 cloud machines and can serve to multiple users in parallel with an average of 1-2 minutes response time.
The entire project was completed with a team of 6.
